# Chat-Server-Multiple-Clients
o run first install the dependencies executing the following command in the 'Chat_Server' folder:
	npm install

Then run:
	node server.js

To try the application open several browser instances at the address:
	http://localhost:3000/chatClient.html

Then chat with each other.

SideNotes: 
  Not allowed space characters when registering names
  Naming is capital sensitive
  if inputting multiple ':' in message more than once will print nothing
